Explanation:

First we create an instance of the class WaterBase. We provide the name of the WaterBase database to use as parameter. If no database exists by that name, a new one is created.

Now let's put some values in our database. WaterBase supports three types of values: Boolean, Integer and String. To save values, use the
db.Save*(key, value) function.

Now if you want to retreive these values, use
db.Load*(key, default_value) function.
